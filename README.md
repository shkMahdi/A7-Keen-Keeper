# 🤝 Keen Keeper

A modern web application designed to help you stay connected with your friends by tracking and managing your interactions. Never lose touch with the people who matter most!

## 📋 Project Description

Keen Keeper is a friendship management application that helps users keep track of their relationships by recording interactions (calls, texts, and video calls) with their friends. The app provides analytics, timelines, and reminders to help maintain meaningful connections with your social circle.

## 🛠️ Technologies Used

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **State Management**: Context API
- **Styling**: Tailwind CSS
- **Data Visualization**: Recharts (Pie Charts)
- **Notifications**: React Hot Toast
- **Loading States**: React Spinners
- **Icons**: React Icons
- **Routing**: React Router

## ✨ Key Features

### 1. **Friend Interaction Tracking**
   - Log interactions with friends (Call, Text, Video Call)
   - Record the date and type of each interaction
   - View detailed information about each friend
   - Toast notifications for successful interactions

### 2. **Analytics Dashboard**
   - Beautiful pie chart visualization showing the distribution of interaction types
   - Real-time statistics updated as you add interactions
   - Visual breakdown of communication methods (text, calls, video calls)
   - Count-based analytics for each interaction type

### 3. **Timeline & Filtering**
   - View all interactions in a chronological timeline
   - Filter interactions by type (Call, Text, Video Call)
   - Formatted date display for easy reading
   - Quick search and organization of interaction history

## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd A7-Keen-Keeper

# Install dependencies
npm install

# Install required packages
npm install react-hot-toast recharts react-spinners
```

### Running the Application

```bash
# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
